Battleship game

This is a simple implementation of the classic Battleship game. The game is designed to be played by one player and bot, and it includes features like placing ships randomly, attacking opponent's ships, and determining the winner.

Binary search tree

Balanced BST(binary search tree) with various features: -insert -delete -find -levelOrder -preorder -inorder -postorder -height -depth -isBalanced -rebalance

Knight travails

This is a solution for the Knight's Travail Problem, which consists of finding a sequence of moves for a knight on a chessboard such that the knight visits every square exactly once.
